# Address book

# Installation

```bash
make up
make provision
```

- Add `symfony.local` to `/etc/hosts`
- Now you can acess to: http://symfony.local/app_dev.php or http://symfony.local/app.php

# Useful commands

Build image
```bash
make build
```

Migrate DB execution
```bash
make migrate
```

Stop and remove containers
```bash
make down
```
